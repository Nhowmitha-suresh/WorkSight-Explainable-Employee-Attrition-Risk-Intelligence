# WorkSight-Explainable-Employee-Attrition-Risk-Intelligence
An AI-driven decision intelligence platform that predicts employee attrition risk and explains key drivers to support HR and business decision-making.
## 🏗 System Architecture

Frontend (React Dashboard)
        ↓
FastAPI Backend
        ↓
Trained Random Forest Model
        ↓
Risk Probability + Business Threshold Layer
        ↓
Explainability Layer (SHAP)
